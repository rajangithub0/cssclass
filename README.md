## Introduction to CSS
CSS (Cascading Style Sheets) is used to style and design web pages. It controls the layout, colors, fonts, spacing, and overall appearance of HTML elements. CSS makes websites look attractive and responsive.

---

## Background Color
The `background-color` property is used to set the background color of an element.

```css
body {
  background-color: lightblue;
}

```
Padding and Margin

Padding: Space inside the element (between content and border)

Margin: Space outside the element (distance from other elements)

```c
.box {
  padding: 20px;
  margin: 15px;
}

```
Background Image

Used to set an image as the background.
```c
.container {
  background-image: url("image.jpg");
  background-size: cover;
}
